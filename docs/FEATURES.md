# 博听 Broaden - 功能创新介绍

> 为听障人士打造的博物馆智能导览应用

## 核心创新

### 1. 双步智能拍摄流程

**问题**：传统博物馆应用无法准确识别展品，用户需要手动搜索或输入展品名称。

**创新**：
- **第一步**：拍摄展牌二维码或文字 → 自动识别展品身份
- **第二步**：拍摄文物主体 → AI 智能提取文物图像（"撕下贴纸"效果）
- **跳过模式**：支持直接拍摄物体，通过视觉 AI 识别

**技术**：iOS Vision Framework + 智谱 AI 视觉模型

---

### 2. 手语数字人解说

**问题**：听障人士无法通过语音导览了解展品，文字说明又不够直观。

**创新**：
- 在线手语数字人实时演示
- 与文字内容同步联动
- 支持简版/详版/手语脚本三种形式切换
- 术语卡片点击查看详细解释

**技术**：WKWebView 集成 + AvatarCoordinator 状态管理

---

### 3. 智能多层级问答

**问题**：静态说明无法满足用户个性化问题。

**创新**：
- **易读版**：简化句式，适合快速理解
- **详细版**：深度解析，满足好奇心
- **手语脚本**：为手语翻译优化的文本
- **继续提问**：AI 持续对话，深入挖掘

**技术**：智谱 AI GLM 模型 + 流式输出

---

### 4. 主体提取技术

**问题**：普通拍照包含太多背景干扰。

**创新**：
- 使用 `VNGenerateForegroundInstanceMaskRequest` 实时生成主体蒙版
- 自动去除复杂背景，只保留文物主体
- 生成透明 PNG 便于后续使用

**技术**：iOS Vision Framework

---

### 5. 跨设备数据同步

**问题**：更换设备后数据丢失。

**创新**：
- iCloud 自动同步展品收藏
- 浏览历史云端保存
- 三重备份：内存 → 本地 → iCloud
- 容错处理，同步延迟时自动合并数据

**技术**：NSUbiquitousKeyValueStore + CloudKit

---

### 6. 无障碍优先设计

**创新**：
- 高对比度配色方案
- 清晰的视觉引导（扫描框、步骤指示器）
- 触觉反馈增强操作确认
- 大字体支持
- 易读版文本优化

---

## 技术亮点

| 技术领域 | 使用技术 | 创新点 |
|---------|---------|--------|
| 图像处理 | Vision Framework | 主体提取、OCR、QR码识别 |
| AI 集成 | 智谱 AI | 展品识别、智能问答、手语翻译 |
| 架构设计 | MVVM + Combine | 清晰的代码分层，易维护 |
| UI 设计 | iOS 26 Liquid Glass | 现代化玻璃拟态设计 |
| 数据同步 | iCloud/CloudKit | 多重备份，容错机制 |
